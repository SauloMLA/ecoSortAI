## Objetivo

Desarrollar una **interfaz funcional y clara** que permita monitorear el estado del sistema, visualizar clasificaciones en tiempo real, y presentar estadísticas de operación.

## Alcance

- Arquitectura backend para recolección de datos
- Dashboard web o local para visualización
- Conectividad IoT entre el sistema embebido y el dashboard
- UX clara para demostración y defensa académica
- Logging de operaciones para análisis
