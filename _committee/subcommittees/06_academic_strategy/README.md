# Subcomité 06 — Estrategia Académica y Defensa

**Estado:** CONSTITUIDO  
**Sesión de creación:** S001  
**Enlace con equipo real:** Documentation Ops

---

## Objetivo

Garantizar que el proyecto sea **defendible académicamente**: bien documentado, con impacto medible, con narrativa coherente, y preparado para preguntas difíciles de un jurado.

## Alcance

- Estrategia de escritura académica
- Análisis de sustentabilidad e impacto
- Medición de impacto cuantificable
- Preparación de presentación
- Enmarcado de innovación
- Simulación de defensa

## Preguntas críticas que debe resolver

1. ¿Cuál es la narrativa central del proyecto?
2. ¿Qué diferencia a EcoSort IA de otros proyectos de reciclaje?
3. ¿Cuáles son las métricas de impacto más convincentes?
4. ¿Qué preguntas difíciles hará el jurado?
5. ¿La documentación técnica soporta escrutinio riguroso?
