# Subcomité 02 — Hardware & Embedded

**Estado:** CONSTITUIDO  
**Sesión de creación:** S001  
**Enlace con equipo real:** Physical Integrator, Software Lead

---

## Objetivo

Lograr un **sistema físico estable y eficiente** que integre cámara, procesador embebido, sensores y actuadores de forma confiable para operación continua.

## Alcance

- Selección de plataforma embebida (Raspberry Pi, Jetson Nano, etc.)
- Configuración de cámara y captura de imagen
- Diseño de circuito de potencia y actuadores
- Integración de sensores auxiliares
- Optimización de latencia del sistema
- Confiabilidad en operación continua

## Preguntas críticas que debe resolver

1. ¿Qué plataforma embebida soporta el modelo de IA con latencia aceptable?
2. ¿Qué cámara ofrece la mejor relación calidad/costo/compatibilidad?
3. ¿Cómo se alimenta el sistema de forma estable?
4. ¿Cuál es la latencia end-to-end del pipeline?
5. ¿El sistema soporta operación continua sin sobrecalentamiento?
