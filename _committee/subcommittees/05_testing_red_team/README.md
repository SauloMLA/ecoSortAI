# Subcomité 05 — Testing & Red Team Global

**Estado:** CONSTITUIDO  
**Sesión de creación:** S001  
**Enlace con equipo real:** QA / Red Team

---

## Objetivo

**Destruir el sistema antes de que falle en la realidad.** Identificar todos los modos de fallo posibles, probar bajo condiciones extremas, y garantizar que el prototipo sea robusto ante uso real.

## Alcance

- Diseño de plan de testing integral
- Pruebas funcionales, de estrés y ambientales
- Análisis de modos de fallo (FMEA global)
- Simulación de mal uso humano
- Pruebas de edge cases
- Evaluación de seguridad básica

## Filosofía

> "Si no lo rompimos nosotros, lo romperá el jurado."

El subcomité opera con la premisa de que **todo lo que puede fallar, fallará**, y su trabajo es encontrar esas fallas antes de la demostración.
