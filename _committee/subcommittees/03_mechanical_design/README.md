# Subcomité 03 — Diseño Mecánico & Materiales

**Estado:** CONSTITUIDO  
**Sesión de creación:** S001  
**Enlace con equipo real:** Mechanical Lead, Physical Integrator

---

## Objetivo

Lograr un **diseño mecánico robusto, libre de atascos**, que permita el flujo continuo de objetos desde la entrada hasta los contenedores de clasificación.

## Alcance

- Diseño del sistema de transporte (banda, rampa, gravedad)
- Diseño del mecanismo de desvío (flaps, servos, aire comprimido)
- Selección de materiales para el prototipo
- Análisis de modos de fallo mecánico
- Diseño para prototipado rápido (impresión 3D, corte láser)
- Prevención de atascos

## Preguntas críticas que debe resolver

1. ¿Qué mecanismo de desvío es más confiable para el rango de objetos?
2. ¿Qué materiales son viables para prototipado rápido y resistentes al uso?
3. ¿Cómo se previenen atascos con objetos irregulares?
4. ¿El diseño permite acceso para mantenimiento?
5. ¿Las tolerancias de impresión 3D son suficientes?
