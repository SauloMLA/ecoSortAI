# Knowledge Base — Base de Conocimiento

> Repositorio centralizado de conocimiento técnico, referencias y estándares del proyecto EcoSort IA.

---

## Estructura

| Carpeta | Contenido |
|---------|-----------|
| `technical_references/` | Papers, datasheets, documentación de componentes |
| `design_standards/` | Estándares de diseño adoptados por el comité |
| `research_notes/` | Notas de investigación de los subcomités |
| `glossary.md` | Glosario técnico del proyecto |

## Convención de nombres

```
[subcomité]_[tipo]_[nombre-descriptivo].[ext]

Ejemplos:
  SC01_paper_mobilenet-v2-classification.md
  SC02_datasheet_raspberry-pi-4b.md
  SC03_standard_3d-printing-tolerances.md
  SC05_note_edge-cases-wet-objects.md
```

## Reglas de uso

1. Todo conocimiento referenciado en sesión debe archivarse aquí
2. Las referencias deben incluir fuente y fecha de acceso
3. Los subcomités son responsables de mantener actualizado su contenido
4. El Artifact Manager cataloga todo lo que ingresa
