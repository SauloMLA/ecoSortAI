# Clerical Staff — Personal Administrativo

> Los agentes administrativos garantizan la integridad documental del comité.  
> No toman decisiones técnicas — mantienen el sistema funcionando.

---

## Miembros

| Rol | Archivo | Función principal |
|-----|---------|-------------------|
| Session Clerk | `SESSION_CLERK.md` | Gestión de registros de sesión |
| Artifact Manager | `ARTIFACT_MANAGER.md` | Organización de artefactos |
| State Keeper | `STATE_KEEPER.md` | Mantenimiento de CURRENT_STATE.md |

## Cuándo actúan

- **Session Clerk** — Al inicio y cierre de cada sesión
- **Artifact Manager** — Cuando se genera un artefacto durante la sesión
- **State Keeper** — Al cierre de cada sesión, después de las decisiones
